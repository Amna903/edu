📊 2. DUAL CODING (Text + Visuals)
What It Is: Every concept taught with both words AND pictures.
How It Works:
Annotated diagrams (not just labeled)
Concept maps showing relationships
Flowcharts for processes
Graphs + equations together
Visual explanations (not walls of text)
Example - Photosynthesis:
Text: "Plants convert CO₂ + H₂O into glucose using light energy"
Visual: Diagram showing chloroplast, light absorption, chemical pathways
Result: You remember both concept AND details
Research: Paivio (1986) demonstrated dual coding improves retention by 50-89% compared to text-only learning.
Status: ✅ Available Now - Integrated throughout all courses

🎯 3. RETRIEVAL PRACTICE (Frequent Quizzes)
What It Is: Regular low-stakes quizzes to strengthen memory (not for grades).
How It Works:
3-5 questions after each concept (micro-quizzes)
10-15 questions at end of topic
Unlimited retakes (focus on learning)
Instant feedback with explanations
Example: After learning Ohm's Law, answer 5 quick questions. Get one wrong? See why, understand the mistake, try similar problems. Memory strengthened.
Research: Roediger & Karpicke (2006) showed testing is more effective than re-reading—improves retention by 50-100%.
Status: ✅ Available Now - Built into every topic

🔀 4. INTERLEAVED PRACTICE (Mix Topics)
What It Is: Practice problems mix different topics together (not blocked).
How It Works:
Instead of 20 "Forces" problems in a row → Mix Forces, Electricity, Waves
Forces student to identify problem type first
Mimics real exams (mixed questions)
Comparison:
Traditional: 20 quadratic equations in a row (boring, superficial)
EduMeUp: Problem 1: Quadratic, Problem 2: Linear, Problem 3: Simultaneous (engaging, deeper)
Research: Rohrer & Taylor (2007) found interleaving improves test performance by 40-70% despite feeling harder during practice.
Status: ✅ Available Now - End-of-chapter quizzes use interleaved practice
