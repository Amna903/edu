# EduMeUp Project

## Overview
EduMeUp is an O-Level & IGCSE Learning Hub built with React, Tailwind CSS, and Express.

## Recent Changes
- Updated **For Parents** page with comprehensive content including O-Level challenges, methodology, and parent partnership modules.
- Updated **For Students** page with detailed solutions to common struggles and the 7-skill learning framework.
- Refactored **Navbar** to include a "Portals" dropdown for Parents, Students, Tutors, and Schools, tidying up the layout and fixing width issues.
- Updated **Pricing** page with a savings calculator and structured program details.
- Updated **Research** page with the 5 pillars of evidence-based learning.
- Updated **Blog** page with a featured article section and categorized grid.
- Fixed dependency corruption by performing a clean `npm install`.
- Moved project files to root for standard Replit environment compatibility.

## Tech Stack
- Frontend: React, Vite, Tailwind CSS, Framer Motion, Lucide React
- Backend: Express, Node.js
- Database: Drizzle ORM (PostgreSQL)
